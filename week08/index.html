Notes

  • Translation functions allow us to move elements left, right, up, or down without impacting the flow of the document because a translated element can only be moved relative to its current position.
  • Scale functions scale an element by the defined factors while maintaining the original aspect ratio. It has no impact on the document flow.
  • Rotation functions rotate an element around the point of origin by a specified angle value. As with scale , by default the point of origin is the elementâ€™s center. Generally, angles are declared in degrees, with positive degrees moving clockwise and negative moving counterclockwise.
  • Transitions allow the values of CSS properties to change over time, essentially defining both starting and ending states.
  • Animations allow us to control each step of an animation via keyframes.
