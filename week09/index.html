Notes

  • We always start using the data- prefix to write attributes, then we write our attribute's name. Finally we can write an optional string value.
    data-powers = 'superSpeed'
    data-max-speed = '10'
  • When we want to access the attributes, we write dataset. and then the attribute's name.
    dataset.powers
    dataset.maxSpeed //we replace the hyphen with camel-case notation
  • Using npm easies the coding with SCSS because we don't need to convert it to CSS to apply our changes.
