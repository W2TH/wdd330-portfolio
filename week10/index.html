Files inside the main folder:
optimusPrime.png
transformers.json          //{name: "Optimus Prime", image: "optimusPrime.png"}

  • Using the json() method
    fetch('transformers.json')
    .then(response => response.json())
    .then(data => {console.log(data)})            //{name: "Optimus Prime", image: "optimusPrime.png"}
  • Using the text() method
    fetch('transformers.json')
    .then(response => response.text())
    .then(data => {console.log(data)})            //"{"name": "Optimus Prime", "image": "optimusPrime.png"}"
  • Using the blob() method
    fetch('optimusPrime.png')
    .then(response => response.blob())
    .then(data => {console.log(data)})            //Blob {size: 29292, type: "image/png"}
