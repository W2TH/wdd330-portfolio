  • Programmer error means the program contains incorrect syntax or faulty logic; it could even be as simple as a typo.
  • User error means the user has entered data incorrectly, which the program is unable to handle.
  • Exception means an error that produces a return value that can then be used by the program to deal with the error. An exception will produce a stack trace.
  • Warning means an error in the code that isn't enough to cause the program to crash; nevertheless, the program will run incorrectly.
