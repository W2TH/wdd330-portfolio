Notes

  • Arrow functions are always anonymous and don't require the return statement.
  • If the function doesn't require any parameters, a pair of empty parentheses must go before the arrow.
    const hello = () => alert('Hello World!')
  • We can use forEach() method in order to loop through any array instead of using the for loop.
    for(let index = 0, max = colors.length ; index < max ; index++ ) //for loop
    colors.forEach( (color,index) => //forEach method()
  • The filter() method returns a new array that only contains items from the original array that return true when passed to the callback.
    const numbers = [ 2, 7, 6, 5, 11, 23, 12 ]
    numbers.filter(x => x%2 === 0 )
    << [ 2, 6, 12 ]
