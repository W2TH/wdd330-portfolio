Notes

  • We can create an empty object by using a constructor function. This method requires less typing and provides a concise way of initializing an object and its properties in one statement.
    const spiderman = new Object()
  • If we want to use curly braces for creating an empty object, the short way requires both the variable name and the property key to be the same.
    //long way
    const ironMan = { name: name, realName: realName };
    //short way
    const ironMan = { name, realName };
  • Permitted values in JSON are double-quoted strings, numbers, true, false, null, arrays and objects.
    const batman = '{"name": "Batman","real name": "Bruce Wayne","height": 74, "weight": 210, "hero": true, "villain": false, "allies": ["Robin","Batgirl","Superman"]}'
  • To generate a random number between 0 and another number, we can multiply the value by that number.
    6 * Math.random(); //generates a random number between 0 and 6
    10 * Math.random(); //generates a random number between 0 and 10

Exercise

function startQuiz() {
const quiz = [
{ name: "Superman", realName: "Clark Kent" },
{ name: "Wonder Woman", realName: "Diana Prince" },
{ name: "Batman", realName: "Bruce Wayne" },
];

const game = {
start(quiz) {
this.questions = [...quiz];
this.score = 0;
// main game loop
for (const question of this.questions) {
this.question = question;
this.ask();
}
// end of main game loop
this.gameOver();
},
ask() {
const question = `What is ${this.question.name}'s real name?`;
const response = prompt(question);
this.check(response);
},
check(response) {
const answer = this.question.realName;
if (response === answer) {
alert('Correct!');
this.score++;
} else {
alert(`Wrong! The correct answer was ${answer}`);
}
},
gameOver() {
alert(`Game Over, you scored ${this.score} point${this.score !== 1 ? 's' : ''}`);
}
}

game.start(quiz);
}
